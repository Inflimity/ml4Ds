# Data Analyst to Data Scientist: The Mastery Path

Welcome to the transition lab! This project is no longer just a single analysis; it is a **5-Level Master Curriculum** designed to transform you into a predictive expert.

## 🚀 How to Start

1.  **Open the Dashboard:**
    Start by opening [CURRICULUM.md](./CURRICULUM.md). This is your "Mission Control" center.
2.  **View Your Mission:**
    Go to the `missions/` directory and open the file for your current level (e.g., [missions/Level_1_Analyst.md](./missions/Level_1_Analyst.md)).
3.  **Use Your Lead Scientist:**
    Your mentor (Antigravity) is no longer here to "discuss" your feelings about the data. They are here to give you **Direct Tactical Instructions**. 
    - Ask: *"I'm ready for Level 1. What's my first task?"*
    - Request validation: *"I've finished Level 1. Am I ready for Level 2?"*

## 🗺️ The Path
- **Level 1:** Foundations & Cleaning (`sales_dataset.csv`)
- **Level 2:** Regression & The Split
- **Level 3:** Feature Architect (`Taxis_Data.xlsx`)
- **Level 4:** Advanced ML (Random Forests)
- **Level 5:** Master Architect (Classification & Deployment)
