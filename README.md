# DA to DS Lab

Welcome to the Data Analyst to Data Scientist transition lab!

In this lab, you will use Antigravity as your Senior Mentor to help you shift your mindset from "What happened?" (Analysis) to "What will happen?" (Science).

## Getting Started

Follow these steps to begin your mission:

1. **Clone the Lab:**
   ```bash
   git clone https://github.com/Musibau-Bamidele/DA-to-DS-Lab.git
   cd DA-to-DS-Lab
   ```

2. **Prepare the Data:**
   The dataset is located in the `data/` folder. Copy the `sales_dataset.csv` into your working codebase directory for training and analysis:
   ```bash
   cp data/sales_dataset.csv ./
   ```

3. **Initialize the Mentor:**
   ```bash
   ag init
   ```

4. **The First Question:**
   Ask your mentor the following exact prompt to kick things off:
   > "I have the sales data. How do I turn this analysis into a machine learning model?"

Happy learning! You provide the data and the goals, and Antigravity provides the patience.
